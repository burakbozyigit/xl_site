# Security Policy

## Scope

ruddyXL is a static website (HTML, CSS, JavaScript) with no backend, no database, and no user authentication. There is no server-side code in this repository.

Contact form submissions are handled by a third-party service ([Formspree](https://formspree.io)). Any vulnerabilities related to that service should be reported directly to Formspree.

## Reporting a Vulnerability

If you believe you've found a security issue in this repository, please email us at **xl@ruddy.ee** rather than opening a public GitHub issue.

We'll do our best to respond within a few days.

## Expectations

Given the static nature of this project, the attack surface is limited. That said, we take reports seriously and appreciate responsible disclosure.
