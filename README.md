# Ruddy XL static multi-page site

Files:
- index.html
- events.html
- gallery.html
- bookings.html
- style.css
- script.js

## Publish on GitHub Pages

1. Create a GitHub repo.
2. Upload all files to the repo root.
3. In GitHub repo settings, open **Pages**.
4. Set the source to your main branch root.
5. Save.

## What to replace first

- venue name if needed
- address
- phone
- email
- Instagram link
- hours
- event descriptions
- demo visuals with real images

## Adding real images

Create an `assets/` folder and replace the gradient placeholders with image files.
Example:

```html
<img src="assets/your-photo.jpg" alt="Ruddy XL crowd" />
```

or change section backgrounds in `style.css`.
